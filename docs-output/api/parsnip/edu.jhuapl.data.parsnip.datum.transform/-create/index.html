open class Create(var fields: MutableList<FieldEncode<*>> = mutableListOf()) : DatumTransform

Computes multiple output fields using DatumTransform instances. These are applied to the input datum to return a new datum with just the fields described by the value encodings. If there are no input fields, keeps all input fields.
